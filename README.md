# 🤖 Бот аттестации сотрудников

Telegram-бот для проведения аттестации с голосовыми ответами, AI-оценкой и Excel-отчётом.

## Как работает

```
Сотрудник → выбирает должность → вводит ФИО
    → отвечает голосом на каждый вопрос
    → Whisper расшифровывает ответ → сотрудник подтверждает
    → GPT-4o оценивает ответ (1-10) + комментарий
    → по итогу генерируется Excel-отчёт → приходит вам в Telegram
```

## Установка

### 1. Получите необходимые токены

| Что нужно | Где получить |
|-----------|-------------|
| Telegram Bot Token | Напишите @BotFather → /newbot |
| OpenAI API Key | https://platform.openai.com/api-keys |
| Ваш Telegram Chat ID | Напишите @userinfobot |

### 2. Клонируйте и настройте

```bash
# Перейдите в папку с ботом
cd attestation_bot

# Установите зависимости
pip install -r requirements.txt

# Создайте файл .env
cp .env.example .env

# Откройте .env и заполните токены
nano .env
```

### 3. Заполните .env

```
TELEGRAM_TOKEN=7123456789:AAF...ваш токен...
OPENAI_API_KEY=sk-proj-...ваш ключ...
ADMIN_CHAT_ID=123456789
```

### 4. Запустите бота

```bash
python bot.py
```

## Как добавить свои вопросы

Откройте `questions.py` и отредактируйте список вопросов для каждой должности:

```python
QUESTIONS = {
    "supervisor": [
        "Ваш вопрос 1...",
        "Ваш вопрос 2...",
        # ...
    ],
    "territory_manager": [...],
    "stationary_merchandiser": [...],
    "sales_rep": [...],
}
```

## Excel-отчёт содержит

- **Лист "Итог"** — ФИО, должность, дата, средняя оценка, результат
- **Лист "Подробные ответы"** — каждый вопрос, расшифровка ответа, оценка, комментарий, рекомендация
- **Лист "Оценки по вопросам"** — таблица с оценками для диаграмм

## Оценки

| Оценка | Результат |
|--------|-----------|
| 8–10 | ✅ Аттестован |
| 6–7.9 | ⚠️ Условно аттестован |
| < 6 | ❌ Не аттестован |
